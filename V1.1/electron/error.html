Server Failed to Start

The AgentCommander backend server could not be reached. This might be a temporary issue or a configuration problem.

Troubleshooting Steps:

  • Make sure Bun is installed on your system
  • Check that port 3001 is not in use by another application
  • Try restarting the application
  • Check the Server menu to manually start/stop the server
  • View server logs with Cmd+` (Mac) or Ctrl+`
