# Document This API

Generate API documentation for this endpoint/service:

Include:
- **Endpoint**: Method + URL
- **Description**: What it does (one line)
- **Authentication**: Required? What type?
- **Request**: Headers, params, body with types
- **Response**: Status codes, body schema, examples
- **Errors**: What can go wrong, error formats
- **Example**: curl command that works

Format as markdown, ready for docs site.
