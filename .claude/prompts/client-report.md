# Client Report

Summarise this work for a client/stakeholder:

1. **What we did**: Plain English, no jargon
2. **Results**: Measurable outcomes
3. **Issues found**: Any problems discovered (with severity)
4. **Recommendations**: What they should do next
5. **Next steps**: What we'll do next

Tone: Professional but friendly. They're not technical.
Length: One page max. Bullet points are fine.
